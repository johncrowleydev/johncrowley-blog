This site is built to be clean, fast, and easy to maintain.

Instead of relying on a heavy CMS, every post lives as a markdown file in the repository. That means writing, versioning, and publishing all happen in one place.

## Why this setup works

- Next.js handles the static generation.
- TypeScript keeps the codebase tidy.
- Markdown keeps writing friction low.
- GitHub Actions deploys the site automatically.

That gives me a straightforward publishing workflow with very little overhead.
